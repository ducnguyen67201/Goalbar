# Contributing

1. Read `AGENTS.md`, `SECURITY.md`, and `docs/architecture.md`.
2. Create a focused feature branch.
3. Never use personal production social accounts for tests.
4. Add deterministic fixtures for agent and platform behavior.
5. Run the validation commands in `README.md`.
6. Include capability and data-retention effects in the change description.

Do not submit credentials, captured private conversations, or unsanitized API responses.
